# PERV Atlas 使用说明

**PERV Atlas** 是猪内源性逆转录病毒（PERV）与相关多组学数据的在线浏览平台。您可以在浏览器中查看统计图表、序列注释、参考基因组定位和多组学信号，并按需下载原始数据或导出当前查看结果。

界面支持 **中文 / English** 切换：点击页面右上角 **「中文」** 或 **「EN」** 即可。

---

## 网站包含哪些模块？

顶部导航对应四个主要页面（首页另含多组学图谱专区）：

| 页面 | 地址 | 做什么用 |
|------|------|----------|
| **首页** | `/` | 总览数据规模、四大模块入口、多组学样本图谱 |
| **概览** | `/overview` | 1,165 条完整 PERV 的统计图与元数据表 |
| **序列 Browser** | `/browser` | 139 条精注释序列的 ORF / 结构域浏览与序列导出 |
| **基因组浏览器** | `/genome` | 猪参考基因组 + 基因注释 + PERV / 同源定位 + 多组学轨道 |

下面按模块说明 **能看什么、怎么操作、能下载什么**。

---

## 首页（`/`）

### 能做什么

- 查看平台简介与核心数据量（完整 PERV 条数、γ/β 分型、精注释条数、同源定位条数等）。
- 通过四张模块卡片进入对应页面。
- 在 **「多组学数据图谱」** 区域：
  - 点击猪体示意图旁的组织卡片或细胞系，查看该组织/细胞系下的多组学文件统计；
  - 切换 **轮播 / 全局概览**，查看发育时期、组织、测序类型等分布图表。

### 能下载什么

首页 **不提供** 数据文件下载；下载请进入下方各功能页面。

## 模块一 · PERV 序列概览（`/overview`）

### 数据内容

- **1,165** 条完整 intact PERV 序列（其中 **1,020** 条 γ.ERV、**145** 条 β.ERV）。
- 配套元数据表，主要字段包括：`Sequence.ID`、`ERV.type`、`Group`（Eastern / Western / Wild）、`Abbretiation`（品种简称）、`Category`、`TE_type`、`Identity`、`Insertion_Time`、`Kimura.distance`、`Motif`、`TSD` 等。

### 能做什么

1. **统计图表**（共 6 张，可交互）  
   - γ.ERV 与 β.ERV 比例  
   - 地理分组（Group）分布  
   - 品种（Abbretiation）分布  
   - Identity、Insertion_Time、Kimura 距离分布直方图  

2. **元数据表**  
   - 按 `Sequence.ID` 搜索  
   - 按 `ERV.type`、`Group` 筛选  
   - 分页浏览  

### 能下载什么（页面底部「数据下载」）

| 文件 | 格式 | 说明 |
|------|------|------|
| `my.final.fa` | FASTA | 全部 1,165 条完整序列 |
| `1165.intact.PERV.infomation.xlsx` | Excel | 与上表对应的元数据 |

点击下载链接即可保存到本地，无需登录。

## 模块二 · 结构和功能注释序列浏览（`/browser`）

### 数据内容

- **139** 条已完成 ORF 与结构域注释的 PERV 序列（`pass.139.fa`）。
- **ORF 注释**：LTR、GAG、POL、ENV（LTR 为非编码区）。  
- **结构域注释**：GAG、AP、RT、RNaseH、INT、ENV。

### 能做什么

1. **左侧列表**：选择序列 ID，或搜索 ID。  
2. **轨道视图**：切换 **ORF** / **结构域** 模式；彩色条块表示各区间，**点击** 某区间载入序列。  
3. **序列显示**：  
   - **DNA**：当前区间的碱基序列  
   - **蛋白**：按标准遗传密码翻译（负链会自动反向互补；选 LTR 时无蛋白，页面会提示改选 GAG/POL/ENV）  
4. **辅助功能**：  
   - **复制**：复制当前显示的 FASTA 文本到剪贴板  
   - **在整条序列中高亮**：展开查看全长序列并标出当前区间  

### 能下载什么

#### A. 页面顶部「原始注释 / 序列文件下载」（整包原始数据）

| 文件 | 格式 | 说明 |
|------|------|------|
| `pass.139.fa` | FASTA | 139 条注释序列全集 |
| `ORF.combine.HTML.bed` | BED6 | 全部 ORF 区间（0-based 坐标） |
| `domin.combine.HTML.bed` | BED6 | 全部结构域区间 |

#### B. 查看序列后的即时导出（工具栏按钮）

| 操作 | 得到什么 |
|------|----------|
| **下载 FASTA** | 当前正在查看的 DNA 或蛋白序列，保存为 `.fa` 文件（文件名随序列 ID 与区间变化） |
| **导出全部蛋白 (FASTA)** | 该序列在 ORF 或结构域模式下 **所有可翻译区间** 的蛋白序列合并为一个 FASTA（ORF 模式自动跳过 LTR） |

## 模块三 · 猪参考基因组浏览器（`/genome`）

> 若管理员尚未在服务器上构建基因组索引，本页会提示「基因组索引尚未构建」，仅模块一、二可用。索引就绪后刷新即可使用。

### 数据与轨道概览

- **参考基因组**：*Sus scrofa* 11.1（Ensembl DNA toplevel）。  
- **基因注释**：Ensembl 108 GTF（浏览器内为 Genes / Transcripts 轨道）。  
- **完整 PERV 在基因组上的位置**：**47** 条（`PERV` 轨道 + 可折叠的「PERV 序列」列表）。  
- **多品种猪 PERV 同源序列**：**876** 条序列，**188** 个 locus（`PERV 同源序列` 抽屉 + `Homologous Seq` / `Homologous Loci` 轨道）。  
- **多组学信号**：来自 `new.Multi-omics/` 目录的 BigWig（`.bw`），在浏览器内加载查看，**不提供整目录打包下载**。

### 能做什么

#### 浏览与定位

- **区间**：选择染色体，输入起止坐标，点 **跳转**。  
- **搜索**：输入 `gene_name`、`gene_id` 或 `transcript_id`，从下拉结果进入对应区域。  
- **IGV 主视图**：缩放、平移；内置轨道包括标尺、Genes、Transcripts、PERV、Homologous 等。  
- **展示模式**：  
  - **Genes** / **TRANSCRIPTS（可变剪切）**：Expanded / Squished / Collapsed  
  - 同一基因的多条转录本可分行对比可变剪切结构  

#### 右侧详情面板

- 点击 **基因 / 转录本 / 外显子**：查看坐标、长度、biotype、外显子序号等。  
- 选中 **转录本** 后进入转录本视图，可 **返回基因** 视图。  
- 点击 **PERV** 或同源列表中的条目：跳转基因组位置，并显示 region、结构域、ORF 等注释（含 **ERV.type** 标签）。  

#### 工具栏常用功能

| 功能 | 作用 |
|------|------|
| **导出 DNA** | 下载 **当前浏览器视窗** 对应区间的参考基因组 DNA（FASTA） |
| **导出 GTF** | 下载该区间重叠的 GTF 注释片段 |
| **多组学轨道** | 打开抽屉，按类别勾选 `.bw` 文件加载到 IGV；可设 **固定 / 自动** 纵轴 |
| **清除多组学** | 一键移除已加载的多组学轨道 |
| **PERV 同源序列** | 打开抽屉：序列视图（筛选、搜索）或 Locus 视图 |
| **恢复轨道** | 若误删了 Genes、Transcripts 等内置轨道，点此恢复 |
| **链颜色** | 为 Genes / Transcripts 设置正链、负链颜色；**联动** 开时为整轨同色 |

#### PERV 序列列表面板

- 展开 **「PERV 序列」**，点击任一条目可定位到该 PERV 在基因组上的位置。

### 能下载什么

#### A. 页面右上角（参考基因组原始文件）

| 文件 | 说明 |
|------|------|
| 参考 FASTA（约 2.4 GB） | 完整 `Sus_scrofa` 11.1 基因组序列 |
| FASTA 索引（`.fai`） | 与上配套的索引文件 |

#### B. 工具栏「导出 DNA / 导出 GTF」

- 导出内容 = **您当前在浏览器里看到的染色体区间**（随跳转、搜索、点击而变化）。  
- DNA 为 FASTA；GTF 为该区间的注释子集文件。

#### C. 「下载可视化」（多组学 + 注释出图）

打开 **下载可视化** 对话框，按步骤操作：

1. **选择基因组区间**（七选一）  
   - 基因、转录本、PERV 序列、同源序列、同源位点、自定义起止坐标、或「单一位点 ± 窗口」  
2. **（可选）上下游延伸**  
   - 在选定区间两侧各延伸若干 bp  
3. **勾选多组学 BigWig 轨道**（至少一条）  
4. **选择渲染方式**  
   - **pyGenomeTracks（推荐）**：所有信号轨合成 **一张图**，可调整颜色、顺序、高度；完成后可 **下载图像** 和 **下载 .ini** 配置文件  
   - **matplotlib**：每条 BigWig **单独一张图**；可多选注释轨叠加；格式 **PDF（默认）/ SVG / PNG**；多文件时自动打 **ZIP**  
5. **（matplotlib 模式）注释轨叠加**  
   - 可选：Genes、Transcripts、PERV、Homologous Seq、Homologous Loci  

**注意**：单次导出区间不能超过 **10 Mb**；过大时系统会提示缩小范围。

#### D. 本模块不提供一键下载的内容

- 876 条同源序列的 **汇总 Excel**、各 **BED 轨道文件**、**BigWig 原始 `.bw`**：请在浏览器内查看，或通过模块一、二下载 PERV 序列与注释；多组学原始文件需向数据提供方另行获取。  
- 右侧详情中的信息：可复制屏幕内容，无单独「导出表格」按钮（同源统计图仅页面内展示）。

## 模块四 · 首页「多组学数据图谱」

### 能做什么

- 了解本平台收录的多组学 BigWig 规模：文件数、样本数、发育时期（如 S、P21、P50…）、组织/细胞系、测序类型（如 ATAC-seq、ChIP-seq、RNA-seq、WGBS 等）。  
- 与 **基因组浏览器 → 多组学轨道** 使用同一套数据；在首页先看统计，再到 `/genome` 加载具体 `.bw` 轨道或 **下载可视化**。

### 能下载什么

- 仅 **统计与图表浏览**，无文件下载入口。  
- 需要图形化导出时，请使用基因组页的 **「下载可视化」**。

## 使用提示

1. **坐标习惯**  
   - 序列 Browser 的 BED 与导出序列为常规生物信息学格式；基因组浏览器内 IGV 为 0-based，详情里标注 **1-based** 的坐标已做显示换算。  

2. **蛋白翻译**  
   - 使用标准遗传密码；终止密码子保留为 `*`；不对读码框做自动校正。  

3. **多组学轨道无信号**  
   - 常见原因是 BigWig 染色体命名与参考基因组不一致（本参考基因组使用 `chr1`–`chr18`、`chrX`、`chrY`、`chrM`）。  

4. **浏览器卡顿**  
   - 参考 FASTA 体积很大，建议优先使用页面内浏览与小范围导出，避免频繁下载全基因组。  

5. **语言**  
   - 切换语言后，部分动态生成的按钮文案会随页面刷新或重新打开抽屉后更新。
